# Project Decisions Log

This file tracks all architectural decisions (ADR style).

---

## Decision Format

- Date
- Decision
- Reason
- Alternatives considered

---

## Example

### 2026-07-01
Use NestJS instead of Express

Reason:
- better architecture
- DI built-in
- scalable structure

Alternatives:
- Express (too low-level)
- Fastify (less structure)
