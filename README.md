# 60 Days Engineering Bootcamp

A 60-day intensive engineering bootcamp focused on becoming a production-ready Fullstack Engineer.

## Goal

Build strong backend and frontend engineering skills while developing production-level software and following professional engineering practices.

This repository contains:

- Daily progress
- Engineering notes
- Architecture decisions
- Code challenges
- Bootcamp projects
- Documentation

---

## Main Technologies

### Backend

- Node.js
- NestJS
- TypeScript
- PostgreSQL
- Redis
- Docker
- RabbitMQ

### Frontend

- React
- Next.js
- TypeScript

### DevOps

- Docker
- Docker Compose
- GitHub Actions (later)
