# 60 Days Bootcamp

This repository is my 60-day intensive backend + fullstack engineering training.

Goal:
Become job-ready Fullstack / Backend Engineer (Node.js + NestJS + TypeScript + React).

Duration:
60 days
- 10 hours/day (weekdays)
- 5 hours/day (weekends)

---

## Stack

Backend:
- Node.js
- NestJS
- TypeScript
- PostgreSQL
- Redis
- Docker

Frontend:
- React
- Next.js

Infrastructure:
- Docker
- CI/CD (later)
- Nginx

---

## Main Project

Public Promise Platform

Users can create public commitments and prove completion (courses, fitness, learning, etc).

## Rules

- Daily commits are mandatory
- No skipping days
- Everything must be versioned in Git
- No copy-paste without understanding
- Every feature must be documented
