# Bootcamp Rules

## Discipline

- 10h/day weekdays
- 5h/day weekends
- daily Git commits

---

## Learning Rules

- If you can't explain it → you don't know it
- No passive watching courses
- Everything must be coded

---

## Code Rules

- TypeScript strict mode ON
- No any types unless justified
- Small functions only
- SOLID principles applied gradually

---

## Workflow

1. Learn concept
2. Implement manually
3. Break it
4. Fix it
5. Document it

---

## Interview Preparation

Every day:
- 3 algorithm tasks
- 1 system design question
- 1 explanation out loud
