# Engineering Handbook

This document defines the engineering standards used throughout the 60 Days Engineering Bootcamp.

The goal is to simulate the workflow of a professional software engineering team.

---

# Principles

- Simplicity over complexity.
- Readability over cleverness.
- Production mindset.
- Continuous improvement.
- Learn by building.

# Definition of Done

A task is considered complete only if:

- Code is implemented.
- Code works.
- Code is committed.
- Documentation is updated.
- The solution can be understood after one month.

---

# AI Usage Policy

AI is a learning assistant.

AI does not replace engineering decisions.

Every generated solution must be understood before use.
